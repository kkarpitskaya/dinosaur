Основной геймплей

Игрок управляет динозавром, который должен перепрыгивать через кактусы, движущиеся слева направо. Цель - набрать как можно больше очков, избегая столкновений.

+5 очков за каждый успешно пройденный кактус
Уровни - повышаются каждые 50 очков (уменьшается расстояние между кактусами и увеличивается скорость)
Жизни - 3 жизни, при столкновении теряется 1 жизнь

Для начала игры...

1)На стартовом экране нажмите кнопку "Старт"
2)Управляйте персонажем с помощью ПРОБЕЛА или клика мыши
3)Избегайте столкновений с кактусами
4)Следите за жизнями - при 0 игра завершается
5)Для рестарта используйте кнопку "Заново"

Элементы управления в игре
Старт/Продолжить - начать игру или продолжить после паузы
Пауза - приостановить игру
Заново - перезапустить игру

Использованная нейросеть - DeepSeek

Время работы - 1,5 часа

Функционал полностью был реализован через нейросеть, правки, которые возникли после ответа, полученного при первом запросе, были внесены самостоятельно
